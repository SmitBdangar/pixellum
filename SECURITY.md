# Security Policy

## Supported Versions

Security updates are provided for supported stable releases.

| Version | Supported |
| ------- | --------- |
| v1.1.0  | :white_check_mark: |
| v2.0.0  | :x: |
| v3.0.0 | :x: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately and do not open a public issue.
