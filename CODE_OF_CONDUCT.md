# Code of Conduct

## Our Commitment

This project is committed to providing a welcoming, respectful, and harassment-free experience for everyone.

## Expected Behavior

Examples of behavior that help create a positive environment include:

- Being respectful in discussions and code reviews
- Giving constructive feedback
- Being open to different viewpoints and experiences
- Taking responsibility for mistakes and learning from them
- Focusing on what is best for the project and community
- Showing empathy toward other contributors

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, discrimination, or personal attacks
- Trolling, insulting, or derogatory comments
- Publishing someone else’s private information without permission
- Repeated disruptive behavior in issues, pull requests, discussions, or other project spaces

## Scope

This Code of Conduct applies within all project spaces, including:

- Issues
- Pull requests
- Discussions
- Project documentation
- Community chat or communication channels related to this project

It also applies when someone is officially representing the project in public spaces.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Maintainers may take any action they consider appropriate in response to behavior that violates this Code of Conduct, including:

- Warning the contributor
- Removing comments, commits, issues, or pull requests
- Temporarily restricting participation
- Permanently banning a contributor from the project community

## Enforcement Guidelines

When reviewing reported behavior, maintainers may consider:

### 1. Correction
Used for minor issues or inappropriate behavior that appears unintentional.

**Possible response:**
- A private warning
- A request to stop the behavior
- Clarification of expected conduct

### 2. Warning
Used for repeated or more serious incidents.

**Possible response:**
- A formal warning
- Temporary restriction from interactions
- Closer review of future participation

### 3. Temporary Ban
Used for serious misconduct or repeated violations.

### 4. Permanent Ban
Used for severe harassment, threats, discrimination, or repeated harmful behavior.

## Attribution

This Code of Conduct was written for the `[pixellum]` project and may be updated over time to reflect the needs of the community.
